================================================================================
REPORT TYPES
================================================================================

1. EXECUTIVE SUMMARY REPORT
   Purpose: High-level overview for executives and decision-makers
   Content:
     - Key findings and metrics
     - Competitive intelligence scores
     - Market analysis summary
     - Company rankings
     - Strategic recommendations
     - Risk assessment
   Best For: Executive presentations, board meetings, strategic planning

2. MARKET ANALYSIS REPORT
   Purpose: Detailed market landscape analysis
   Content:
     - Market overview statistics
     - Disease area analysis
     - Clinical phase distribution
     - Partnership analysis
     - Technology landscape
     - Financial metrics
     - Competitive positioning
   Best For: Market research, competitive analysis, investment decisions

3. COMPANY PROFILES REPORT
   Purpose: Detailed profiles of all companies in filtered dataset
   Content:
     - Stock information (symbol, price, market cap)
     - Business information (disease area, products, technology)
     - Pipeline information (active trials, partnerships)
     - Individual company analysis
   Best For: Company research, due diligence, partnership evaluation

4. CLINICAL TRIAL ANALYSIS REPORT
   Purpose: Comprehensive clinical trial data analysis
   Content:
     - Trial overview and statistics
     - Trial status breakdown
     - Study type distribution
     - Disease focus analysis
     - Clinical phase analysis
     - Trial timeline information
   Best For: Clinical research, regulatory analysis, trial tracking

5. COMPETITIVE INTELLIGENCE REPORT
   Purpose: Strategic competitive analysis
   Content:
     - CI domain scores (6 domains)
     - Overall risk index
     - Competitive positioning
     - Market leaders ranking
     - Strategic recommendations
     - Market outlook
   Best For: Strategic planning, competitive positioning, risk assessment

2. CSV FORMAT (.csv)

   - File size: Medium

HOW TO USE REPORT GENERATION
================================================================================

STEP 1: APPLY FILTERS
  1. Go to "Input & Filters" tab
  2. Select desired filters:
     - Disease Area
     - Indication
     - Clinical Phase
     - Geography
     - Market Cap
     - Trial Status
  3. Filters are applied automatically

STEP 2: NAVIGATE TO REPORTS TAB
  1. Click on "Reports" tab
  2. You'll see the report generation interface

STEP 3: SELECT REPORT TYPE
  1. Choose from 5 report types:
     - Executive Summary
     - Market Analysis
     - Company Profiles
     - Clinical Trial Analysis
     - Competitive Intelligence

STEP 4: GENERATE REPORT
  1. Click "Generate Report" button
  2. Wait for report to be generated
  3. Report content will be displayed

STEP 5: EXPORT REPORT
  Choose export format:
  - Download as TXT (plain text)
  - Download Data as CSV (spreadsheet)
  - Download as Excel (formatted workbook)

STEP 6: BATCH GENERATION (Optional)
  1. Click "Generate All Reports" button
  2. All 5 reports will be generated
  3. Download combined TXT file or Excel workbook

EXPORT FILE SPECIFICATIONS
================================================================================

TEXT FILE (.txt):
  - Format: Plain text
  - Encoding: UTF-8
  - Line breaks: CRLF (Windows)
  - File naming: {ReportType}_{YYYYMMDD_HHMMSS}.txt
  - Example: Executive_Summary_20260404_050130.txt

CSV FILE (.csv):
  - Format: Comma-separated values
  - Encoding: UTF-8
  - Delimiter: Comma (,)
  - Quote character: Double quote (")
  - File naming: clinical_trials_{YYYYMMDD_HHMMSS}.csv
  - Contains: All filtered data records

EXCEL FILE (.xlsx):
  - Format: Excel 2007+ (.xlsx)
  - Encoding: UTF-8
  - Multiple sheets:
    1. Clinical Trials (all filtered data)
    2. Summary (key metrics)
    3. CI Scores (domain scores)
    4. Companies (company summary)
  - File naming: pci_report_{YYYYMMDD_HHMMSS}.xlsx
  - Batch file naming: all_data_{YYYYMMDD_HHMMSS}.xlsx

BATCH REPORT GENERATION

  • Excel workbook with all data and summaries

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: Report generation is slow
SOLUTION:
  • Reduce filtered dataset size
  • Apply more specific filters
  • Use single report instead of batch

ISSUE: Report content is incomplete
SOLUTION:
  • Verify filters are applied correctly
  • Check that data is loaded
  • Regenerate report
  • Check browser console for errors

ISSUE: Export file is not downloading
SOLUTION:
  • Check browser download settings
  • Disable pop-up blockers
  • Try different browser
  • Try different export format

ISSUE: Excel file won't open
SOLUTION:
  • Ensure Excel is installed
  • Try opening with different application
  • Check file is not corrupted
  • Regenerate report
